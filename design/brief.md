# Game Brief — Crystal Vanguard（仮）

## 実行環境（エンジン）

- エンジン: unity
- 次元: 3D（unity・unreal）
- 選択理由: ハーネスの unity 3D パイプライン（Meshy 3D 資産・Menu 必須・メタ進行）の評価ラン（E2）として人間が指定

## ゲーム像

3D アリーナ・ウェーブサバイバー。プレイヤーは1体のヒーローとなり、円形アリーナの中心付近で四方から迫るクリーチャーの群れを自動攻撃で薙ぎ払いながら、ダッシュ回避で紙一重の立ち回りを続ける。倒した敵はクリスタルを落とし、ウェーブが進むごとに敵は速く・多くなる。Vampire Survivors 系の「群れを捌く快感」を 3D の最小構成（1アリーナ・敵1〜2種・キャラ1体）に凝縮した点が参照作品との差分。

## 遊び手とセッション長

自分と友人がデスクトップ（macOS ビルド）で遊ぶ。1セッション 2〜3分。リトライは Result から入力1回で即時。ラン間はメニューでアップグレードを購入して再挑戦する「もう一回」ループを回す。

## Core Fantasy

迫る群れを紙一重で躱しながら自動攻撃で薙ぎ払う無双感。「避ける判断」に集中させ、攻撃は自動化して照準負荷をゼロにする。

## 参照作品

| 作品 | 借りる要素 | 借りない要素 |
|---|---|---|
| Vampire Survivors | 自動攻撃・群れを捌く快感・ラン間アップグレード | 武器進化ツリー・多種武器・ステージ複数 |
| Risk of Rain 2 | 3D空間での回避移動の緊張感 | ローグライクアイテム・ボス・マップ探索 |
| Devil Daggers | 1アリーナ・エンドレス・スコアアタックの純度 | FPS視点・手動照準 |

## 操作

- WASD / 矢印キー: 移動
- Space: ダッシュ回避（短い無敵時間つき・クールダウンあり）
- 攻撃: 自動（最寄りの敵へ周期発動。照準・攻撃ボタンなし）
- 同時押しは移動+Space のみ。マウス不要（Menu 操作もキーボードで完結）

## 勝敗

- 勝利条件: なし（エンドレス）。
- 敗北条件: HP が 0 になった時点でラン終了 → Result へ。
- スコア: あり（生存時間 + 撃破数由来のスコア + 獲得クリスタル）。ハイスコアを永続記録する。

## アウトゲーム / やり込み

- 選択要素（2つ以上）: 通貨（クリスタル） + 持ち越しアップグレード
- 志向の1文: ランで稼いだクリスタルを Menu でアップグレード（攻撃力・移動速度・HP）に変換し、「今回はもっと先のウェーブへ」という強くなって再挑戦する動機を作る（ハイスコア+統計は必須要素として当然含む）

## スコープ制約

- 実装時間の想定: 数時間以内（自律実装）
- 画像資産数の上限: 12点以内（key image・UI・コンセプト画含む）
- 音資産数の上限: SFX 6点 / BGM 1曲（ループ）
- 3Dモデル数の上限: 4点以内（hero 1・敵 1〜2・プロップ 1）+ アニメクリップは hero 3（idle/run/attack）+ 敵 1〜2
- 予算: state/budget.txt に従う（$100。見積もり $6〜10）
- 盛らない宣言: 敵3種目・ボス・複数アリーナ・武器種追加・スキン/実績の追加提案は却下する

## アート方向

- ピクセルアート: no（3D）
- 方向性: スタイライズド・トゥーン。明るいファンタジー基調にネオン風のアクセント（クリスタルの発光色）。フラット寄りの塗りでシルエット可読性を最優先（hero=縦長・敵=低く横に広い・クリスタル=幾何学形状）。ポリゴン予算は assets-config.md 既定（hero ≤ 50k tri / prop ≤ 10k tri / 環境 ≤ 100k tri）

## 音の方向

BGM はドライビングなエレクトロ（BPM 固定・シームレスループ 60〜90秒・インスト）。SFX はパンチの効いた明快な電子音系（攻撃ヒット・ダッシュ・被弾・クリスタル取得・ウェーブ開始・購入）。ElevenLabs starter のサブスク枠内で生成し、枠不足時は縮退を報告する（黙って置換しない）。
