# Code of Conduct

ArcadeRelay aims to be a practical, respectful project for people building and
reviewing AI-assisted game tooling.

## Expected Behavior

- Be direct, constructive, and specific.
- Assume good intent while still requiring evidence for technical claims.
- Respect contributor time by using issue and pull request templates.
- Keep critique focused on the work, not the person.
- Disclose conflicts of interest when provider, license, or asset-generation
  recommendations may be affected.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive or discriminatory language.
- Publishing private contact details, secrets, credentials, or private reports.
- Repeated off-topic arguments that block project work.
- Misrepresenting generated assets, licenses, QA evidence, or security impact.

## Enforcement

Maintainers may edit or remove comments, close issues, reject pull requests,
limit participation, or block contributors when behavior harms the project.

For public moderation concerns, open an issue and keep the report factual. For
private or sensitive concerns, contact a maintainer through a private GitHub
channel when available. Do not include private personal information in public
issues.
